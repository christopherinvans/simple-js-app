# Pokédex App

Web app that pulls data from [PokeApi](https://pokeapi.co/) to display info on the original 150 Pokémon.

##Technologies Within

HTML5
CSS3
JavaScript
jQuery 3.3.1
Bootstrap 4.3.1
ESLint
Prettier
